What is JWT?

JWT (JSON Web Token) is a compact, URL-safe means of representing claims between two parties.

It's commonly used for authentication and authorization in web applications.

How It Works

1. User logs in with credentials

2. Server validates credentials and issues a signed JWT

3. Client stores the JWT and includes it in subsequent requests

4. Server validates the JWT signature to authenticate the user

JWT Structure

Header: Contains token type and signing algorithm

Payload: Contains claims (user data, expiration)

Signature: Verifies the token hasn't been altered
